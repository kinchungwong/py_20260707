## How it was built (workflow)

A 7-phase multi-agent workflow (20 agents, ~30 min, 0 errors): build foundation →
build audio → author tests (fan-out) → **10 adversarial verifiers** → integrate →
self-heal loop → completeness critic. Notable: it **built clean on the first pass
(0 fix iterations)**; every verifier came back `refuted:false` (incl. a bit-exact
AMP-move proof and a 528 Hz non-12-TET probe showing no note-grid snapping). The
critic found only new-surface coverage gaps, closed in a focused follow-up (+13 tests).
